## Instrucciones

1. **Habilitar el entorno virtual:**

```bash
    python -m venv .venv
```

2. **Acceder al entorno virtual:**

- **Windows**
```bash
   venv\Scripts\activate
```

- **Linux / MacOS**

```bash
   source venv/bin/activate
```

3. **Descargar dependencias**

```python
    pip install -r requirements.txt
```
